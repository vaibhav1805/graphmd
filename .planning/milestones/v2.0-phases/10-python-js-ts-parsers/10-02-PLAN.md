  <behavior>
    The JSParser implements the LanguageParser interface (Name/Extensions/ParseFile).

    Import tracking (four regex patterns for dual module system):
    - ESM default: `import axios from 'axios'` -> importMap["axios"] = "axios"
    - ESM named: `import { Pool } from 'pg'` -> importMap["Pool"] = "pg"
    - CommonJS default: `const axios = require('axios')` -> importMap["axios"] = "axios"
    - CommonJS destructured: `const { Pool } = require('pg')` -> importMap["Pool"] = "pg"

    Detection patterns (declarative, following GoParser template):
    - HTTP: axios.get/post/put/delete/patch/request (0.9), fetch() as bare global call (0.9), got.get/post (0.9)
    - DB: pg Pool/Client constructor (0.85), mysql2 createConnection/createPool (0.85), mongoose.connect (0.85), mongodb MongoClient (0.85), prisma PrismaClient constructor (0.85), sequelize Sequelize constructor (0.85)
    - Cache: ioredis Redis constructor (0.85), redis createClient (0.85)
    - Queue: kafkajs Kafka constructor (0.85), amqplib connect (0.85), @aws-sdk/client-sqs SQSClient constructor (0.85)

    Comment hints: `// Calls payment-api`, `// Depends on auth-service` -> confidence 0.4

    False positive control:
    - Skip lines starting with // (single-line comments)
    - Track inBlockComment state for /* ... */ block comments
    - Skip test files: *.test.js, *.test.ts, *.spec.js, *.spec.ts, *.test.jsx, *.test.tsx
    - No import-only signals (require usage evidence)

    Special cases:
    - fetch() is a global — no import required. Match bare `fetch(` calls with URL argument.
    - Destructured imports: `import { get } from 'axios'` means bare `get(...)` maps to axios package.
    - `new Pool(...)`, `new Client(...)`, `new Redis(...)` — match `new Constructor(...)` pattern for constructors.

    Target extraction: Parse URL from first string argument (single/double quotes and backtick template literals) using net/url, extract hostname.

    Test cases (inline JS/TS source fixtures):
    - axios.get("http://payment-api:8080/pay") -> http_call, target=payment-api
    - const axios = require('axios'); axios.post("http://user-svc/create") -> http_call, target=user-svc
    - fetch("http://order-api:3000/orders") -> http_call, target=order-api (no import needed)
    - import { Pool } from 'pg'; new Pool({host: "primary-db"}) -> db_connection signal
    - mongoose.connect("mongodb://primary-db:27017/mydb") -> db_connection, target=primary-db
    - import Redis from 'ioredis'; new Redis("redis://cache-01:6379") -> cache_client, target=cache-01
    - import { Kafka } from 'kafkajs'; new Kafka({brokers: ["kafka-01:9092"]}) -> queue_producer signal
    - amqplib.connect("amqp://rabbitmq-host") -> queue_producer, target=rabbitmq-host
    - import only (import axios from 'axios' with no call) -> no signals
    - single-line comment (// axios.get("http://old")) -> no signals
    - block comment (/* fetch("http://test") */) -> no signals
    - test file (handler.test.ts) -> no signals
    - // Calls payment-api -> comment_hint signal at 0.4 confidence
    - ESM + CommonJS in same file both detected
  </behavior>
  <implementation>
    **patterns.go:** Define JSDetectionPattern struct (Package, Function, Kind, TargetType, Confidence, ArgIndex, IsConstructor bool, IsBareCall bool). Create DefaultJSPatterns slice. Define import regex patterns (4 regexes: ESM default, ESM named, CJS default, CJS destructured). Define call/constructor/bare-call regexes and comment hint regex.

    **parser.go:** JSParser struct with patterns map and bareCallPatterns slice. NewJSParser() constructor. Name() returns "javascript". Extensions() returns [".js", ".ts", ".jsx", ".tsx"]. ParseFile():
    1. Skip test files (check filename patterns: *.test.*, *.spec.*)
    2. Split content into lines
    3. First pass: build importMap from import/require statements using regexes. Track both default and destructured imports.
    4. Second pass: for each line, skip if in block comment or single-line comment. Match:
       a. package.function() calls against importMap + patterns (like GoParser)
       b. bare function calls (fetch, destructured imports) against importMap + patterns
       c. `new Constructor(...)` calls against importMap + constructor patterns
    5. Extract URL target from first string arg (handle single quotes, double quotes, backticks) using net/url
    6. Scan for comment hints matching `//\s*(?:Calls|Depends on|Uses|Connects to)\s+(\S+)`
    7. Return []CodeSignal with Language="javascript"

    Follow GoParser's evidenceSnippet pattern. Compile-time interface check: `var _ code.LanguageParser = (*JSParser)(nil)`.
  </implementation>
