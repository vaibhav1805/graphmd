Create `internal/knowledge/signal_convert.go` with these functions:

1. `convertCodeSignalsToDiscovered(signals []code.CodeSignal, sourceComponent string) []*DiscoveredEdge` — Converts CodeSignal values into DiscoveredEdge values for the merge pipeline:
   - Group by (sourceComponent, targetComponent, edgeType) key using null-byte separator
   - For each unique key, create a DiscoveredEdge with:
     - Edge via `NewEdge(source, target, edgeType, highestConfidence, bestEvidence)`
     - `Edge.ExtractionMethod = "code-analysis"`
     - `Edge.SourceFile = sig.SourceFile` (from highest confidence signal)
     - `Edge.SourceType = "code"`
     - Signals slice with one Signal per input signal: `{SourceType: SignalCode, Confidence: sig.Confidence, Evidence: sig.Evidence, Weight: AlgorithmWeight["code"]}`
   - Map `DetectionKind` to EdgeType: all code detection kinds map to `EdgeDependsOn` for simplicity (research recommendation). The fine-grained type info lives in code_signals table.
   - Skip signals where source == target (self-loops) or target is empty
   - Return deterministic order (sort by key)

2. `applyMultiSourceConfidence(edges []*DiscoveredEdge)` — Post-merge probabilistic OR:
   - For each edge, scan its Signals to find max confidence from markdown sources (any SourceType != SignalCode) and max confidence from code sources (SourceType == SignalCode)
   - If BOTH markdown and code signals present: `merged = 1.0 - (1.0-mdConf)*(1.0-codeConf)`, clamped to min(merged, 1.0)
   - Update `de.Confidence` and `de.Edge.Confidence` with merged value
   - If only one source type: leave confidence unchanged (already correct)

3. `determineSourceType(de *DiscoveredEdge) string` — Returns "markdown", "code", or "both":
   - Scan Signals for SignalCode and non-SignalCode entries
   - Return "both" if both present, "code" if only code, "markdown" otherwise

4. `ensureCodeTargetNodes(g *Graph, codeEdges []*DiscoveredEdge)` — Creates stub nodes for code-detected targets not already in graph:
   - For each edge's Target, check if `g.Nodes[target]` exists
   - If not, `g.AddNode(&Node{ID: target, Type: "infrastructure"})` with ComponentType derived from the code signal's TargetType if available
   - Also check Source (sourceComponent may not be a markdown node); create stub if needed
